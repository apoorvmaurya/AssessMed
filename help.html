How to go about AssessMed?

Please find below the Steps for Administrators to go about the AssessMed assesment System. Click to Learn More

1. Log into the System using your account to start generating AssessMed assesments
2. Select the type of disease for which you want to generate risks
3. Input the Patient's Vitals
4. Click on Submit
5. Get Assessment Result
6. Download as PDF
